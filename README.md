# Zarpen entdecken

Testkarte für die Tour „Redder & Hundewiese“.

Enthalten:
- OpenStreetMap mit Leaflet
- farbliche Route
- eigene Marker per CSS
- Popups
- Infobox und Legende

Hinweis: Der Tourverlauf ist aktuell anhand der gesetzten Punkte grob nachgezeichnet. Später sollte er durch eine exakte GPX-Datei aus Komoot ersetzt werden.
